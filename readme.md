🌾 Direct Market Access for Farmers

Problem:
Farmers lose money due to middlemen and don’t get fair prices.

💡 Your Project Idea: Krushi Settu (Smart Farmer Marketplace)
🎯 Main Idea

Build a platform where:
👉 Farmers can sell directly to buyers
👉 Buyers can purchase at fair prices
👉 No middlemen → more profit for farmers

🧠 Core Solution Concept
🧑‍🌾 For Farmers:
Upload crop details (rice, wheat, vegetables)
Set their own price
See real-time market price
Get direct orders from buyers
🛒 For Buyers:
Browse nearby farmers
Compare prices
Buy fresh products directly
Get better deals than market
🔥 Key Features (Very Important for Hackathon)
1. 📍 Location-Based Matching
Show nearby farmers
Reduce transport cost
2. 📊 Real-Time Market Price
Show mandi price
Help farmers decide fair price
3. 🤝 Direct Deal System
Chat between farmer & buyer
Negotiate price
4. 🛒 Simple Marketplace UI
Product cards (crop name, price, quantity)
“Buy Now” button
5. 🔐 Aadhaar Login (your idea 👍)
Farmer verification
Trusted platform
6. 📦 Order Tracking
Buyer can track order
Farmer can see order
